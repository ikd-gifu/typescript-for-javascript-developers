現在チェックアウト中のローカルブランチのコミットをリモートリポジトリ（origin）の同名ブランチにプッシュしつつ、上流ブランチ（upstream）を設定する
git push -u origin HEAD

HEADの意味
HEADは現在作業中のブランチを指すポインタで、このコマンドではブランチ名を指定せずにカレントブランチをプッシュします。
​
リモートに同名ブランチが存在しない場合でも、新規作成されます。
​
-uオプションの効果
-u（--set-upstream）は、プッシュ後にローカルブランチとリモートブランチを紐づけ、次回からのgit pushやgit pullを省略形（ブランチ名なし）で実行可能にします。
​
初回プッシュ時に特に便利で、例えばgit push -u origin HEADを実行後、git pushだけで済みます。

git merge - 
「直前に checkout していたブランチ」をマージ元として、現在のブランチに取り込むためのショートカット

git push origin HEAD
「今いるローカルブランチを、origin 上の同名ブランチにプッシュする」コマンドです。
